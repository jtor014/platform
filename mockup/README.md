# Mockup

Open platform-mockup.html in a browser to see the design direction.

## Important

This mockup is a VISUAL REFERENCE ONLY. It shows the aesthetic direction — colours, typography, component patterns, page feel.

It is NOT a structural authority. Do not use it to determine:
- Routes (use .claude/routes.md)
- Content (use content files with frontmatter)
- Navigation structure (use .claude/routes.md)
- Component architecture (use CLAUDE.md)
- Design tokens (use .claude/design-rules.md)

The mockup was built before the architecture was finalised. Some values differ from the design system (font choices, exact colours, widths). The design-rules.md file is correct. The mockup shows the vibe.
